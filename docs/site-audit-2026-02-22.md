# Аудит сайта okoroleva.com (22.02.2026)

Короткий аудит по коду и структуре страниц с фокусом на ошибки, дизайн-конверсию и SEO для сайта психоаналитика.

## 1) Что проверено

- Базовая техническая целостность HTML-страниц в `docs/`.
- Наличие обязательных SEO-элементов: `title`, `meta description`, `canonical`, Open Graph.
- Наличие и корректность структурированных данных (`LocalBusiness`, `Person`, `FAQPage`, `Service`, `BlogPosting`).
- Структура контента и UX-паттерны (CTA, доверительные блоки, навигация, путь до записи).

## 2) Ошибки и риски

### Критичных технических ошибок не обнаружено

По быстрым проверкам страницы содержат базовые SEO-элементы и валидную структуру для индексации.

### Замечания (низкий/средний приоритет)

1. **`meta keywords` всё ещё используется на ключевых страницах**.
   Поисковые системы (Google/Яндекс) практически не используют этот тег как фактор ранжирования. Это не ошибка, но «шум» в head.

2. **Open Graph использует SVG как превью (`og:image`)**.
   Для соцсетей и мессенджеров стабильнее PNG/JPG 1200×630. На части платформ SVG в превью может отображаться нестабильно.

3. **Позиционирование в заголовках немного смешано**: на главной акцент на «нейропсихолог», а в услугах/текстах — «психоаналитическая терапия».
   Для SEO и конверсии лучше единая «основная роль» на уровне кластера страниц и query intent.

4. **Контент-кластер блога пока небольшой**.
   Блог качественный, но для устойчивого роста органики по психотерапевтическим запросам нужен более плотный календарь публикаций.

## 3) Рекомендации по дизайну (с оглядкой на лучшие сайты практикующих психоаналитиков)

1. **Усилить блок доверия рядом с первым экраном**:
   - «Стаж / образование / членство в проф. сообществах / супервизия».  
   - Формат: 3–4 коротких маркера + ссылка «Подробнее обо мне».

2. **Добавить микро-сценарии выбора услуги**:
   - «Если у вас тревога / выгорание / сложности в отношениях — начните отсюда».  
   Это повышает понятность маршрута и снижает когнитивную нагрузку.

3. **Укрепить доказательность без нарушения этики**:
   - Блок «Как строится работа: 1-я встреча → гипотеза → план → периодическая оценка динамики».  
   У лучших сайтов психологов это часто ключевой конверсионный элемент.

4. **Добавить отдельный CTA после каждого смыслового блока** (не только в hero и футере):
   - «Задать вопрос», «Уточнить формат», «Записаться на первую встречу».

## 4) Рекомендации по SEO-продвижению

1. **Семантические кластеры под коммерческий и информационный интент**:
   - Коммерческие: «психоаналитик москва», «психоаналитик онлайн», «психотерапия при тревоге москва».  
   - Информационные: «как понять что нужна психотерапия», «чем психоанализ отличается от КПТ», «как выбрать психоаналитика».

2. **Расширить landing-pages под запросы услуг**:
   - Отдельные страницы: тревога, выгорание, отношения, самооценка, кризисы.  
   - Для каждой: симптомы, как проходит работа, ожидаемая динамика, FAQ, мягкий CTA.

3. **Усилить E-E-A-T сигналы**:
   - Полная проф-биография, подтверждающие документы, членство, супервизия, этический кодекс, условия конфиденциальности.  
   - Указать автора и дату обновления на всех экспертных страницах.

4. **Контент-план на 3 месяца (минимум 2–4 статьи/мес.)**:
   - Каждая статья должна вести на релевантную услугу + блок записи.
   - Добавить внутреннюю перелинковку «похожие материалы».

5. **Техническое SEO**:
   - Проверить Rich Results/Schema Validator после каждого значимого обновления.
   - Добавить контроль Core Web Vitals (LCP/CLS/INP) в Search Console и Яндекс.Вебмастере.

## 5) Что делать в первую очередь (приоритет)

1. Перевести `og:image` на PNG/JPG 1200×630.
2. Удалить `meta keywords` со всех страниц.
3. Определить единый основной интент бренда в заголовках (`психоаналитик` vs `нейропсихолог`) и синхронизировать Title/H1 на ключевых страницах.
4. Запустить контент-календарь из 8–12 статей на 3 месяца с внутренней перелинковкой на услуги.
